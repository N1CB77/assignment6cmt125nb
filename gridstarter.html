CSS Grid without text-align

CSS Grid

I added display:grid; to the articles. display:grid; activates the grid elements. Without it the grid elements would not work. Then I used grid-template-columns: which makes changes to the number of columns stated. I put repeat so it would make the same changes to every column availible. I put auto-fill which is similar to flex-wrap, all of the boxes will fill in the webpage and flow from ledt to right. minmax means the size range that the boxes will change. I used 19rem which is the minimum width of columns shown in a row and I used 1fr (fraction) which is how much space the boxes will fill in remaining space. For the sections I put the width as 90% because when I used 100% the boxes did not fully fit inside of the webpage because I had used grid-gap in the articles which adds somes space between the columns and added on to the original width, so making the width 90% made the boxes smaller making it fully fit inside the webpage.
